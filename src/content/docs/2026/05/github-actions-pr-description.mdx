A step-by-step guide to building a GitHub Action that uses a Mastra agent to automatically write pull request descriptions. It covers wiring up a Mastra agent with an LLM, configuring the workflow to trigger on pull requests, and having the agent analyse the diff to produce a clear, useful PR description.
