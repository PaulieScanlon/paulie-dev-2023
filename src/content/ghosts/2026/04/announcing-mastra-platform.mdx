Today we're launching the Mastra platform with tools to develop, run, manage and monitor your agents effectively at scale — featuring Mastra Studio, Mastra Server, and Memory Gateway.

These are three separate products, each a game changer in its own right. Studio provides evals, logs, traces, datasets, and metrics. Server handles deploying agents and workflows. Memory Gateway delivers state-of-the-art agent memory.
