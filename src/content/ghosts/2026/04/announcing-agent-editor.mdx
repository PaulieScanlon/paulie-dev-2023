Mastra Studio now has an Agent Editor that lets subject matter experts and product teams iterate on agent behavior from Studio, without touching code or redeploying.

Developers continue defining agents in code as usual, but the editor stores modifications separately using a draft/publish workflow. This enables version comparison, rollback capabilities, and maintains a complete history of agent configuration changes.
