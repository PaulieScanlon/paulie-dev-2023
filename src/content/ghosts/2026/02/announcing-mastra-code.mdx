An open-source TUI coding agent with observational memory that compresses context without losing it.

Mastra Code is a TUI coding agent that never forgets. It's powered by observational memory, which watches your conversation, generates observations, and reflects on them to compress context without compacting and losing important information.

About half the team at Mastra already use it as a daily driver. Some have had sessions running for months, maintaining useful knowledge without losing what was learned.
