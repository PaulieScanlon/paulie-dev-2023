In this post I'll explain how I've created an excerpt from Markdown content. I've used this approach on my site, I'm not sure if it's the best way to do it, but it works!

## What is an Excerpt?

To quote [Wikipedia](https://en.wiktionary.org/wiki/excerpt) an excerpt is:

> A clip, snippet, passage or extract from a larger work such as a news article, a film, or a literary composition.

An excerpt is helpful in "preview cards" like I have on my [/posts](/posts) page. It gives the reader a little more context about the post before clicking the link. They usually look like this, where ellipses are used to denote truncated text.

## Astro Content Collections

If you're using [Content Collections](https://docs.astro.build/en/guides/content-collections/), the following code snippet should look familiar.

```javascript
---

const posts = await getCollection('posts');
---

<ul>
  {

      console.log(post.body)
      return (
```

## Covert HTML string

To convert the HTML into something you can use you'll need map over each of the HTML element in the array, strip out all the opening and closing HTML tags, and return the bit in the middle, the text.

I've accomplished this with the below `createExcerpt` helper function, and some [regex](https://en.wikipedia.org/wiki/Regular_expression) written by ChatGPT ðŸ’….

```diff
---
import MarkdownIt from 'markdown-it';
import { getCollection } from 'astro:content';

const posts = await getCollection('posts');
---

<ul>
  {
    posts.map((post) => {
+      const parser = new MarkdownIt();
-      console.log(parser.render(post.body));

+      const createExcerpt = (body) => {
+        return parser
+          .render(body)
+          .split('\n')
+          .slice(0, 6)
+          .map((str) => {
+            return str.replace(/<\/?[^>]+(>|$)/g, '').split('\n');
+          })
+          .flat()
+          .join(' ');
+      };

+      const excerpt = createExcerpt(post.body);
+      console.log(excerpt);

      return (
        <li>
          <a href={post.slug}>
            <strong>{post.data.title}</strong>
            <p>Excerpt goes here...</p>
          </a>
        </li>
      );
    })
  }
</ul>
```

## Creating the substring

Whilst the above string is nearly what I wanted, it's not exactly what I wanted so I added one final finishing touch.

```diff
---
import MarkdownIt from 'markdown-it';
import { getCollection } from 'astro:content';

const posts = await getCollection('posts');
---

<ul>
  {
    posts.map((post) => {
      const parser = new MarkdownIt();

      const createExcerpt = (body) => {
        return parser
          .render(body)
          .split('\n')
          .slice(0, 6)
          .map((str) => {
            return str.replace(/<\/?[^>]+(>|$)/g, '').split('\n');
          })
          .flat()
          .join(' ');
      };

-     const excerpt = createExcerpt(post.body);
+     const excerpt = `${createExcerpt(post.body).substring(0, 140)}...`;
      console.log(excerpt);

      return (
        <li>
          <a href={post.slug}>
            <strong>{post.data.title}</strong>
            <p>Excerpt goes here...</p>
          </a>
        </li>
      );
    })
  }
</ul>

```

The above change can be configured to truncate the text at a given character count and by adding a template literal, I've added on the all important `...` to the end. The final output now looks like the below.

## Final Usage

In my site I have multiple collections, so I've reused this function in a number of collections, as such, I abstracted the function into my `utils` directory. I can then import and use it wherever it's needed.

Here's the final code.

```javascript
// src/utils/create-excerpt

import MarkdownIt from 'markdown-it';
const parser = new MarkdownIt();

export const createExcerpt = (body) => {
  return parser
    .render(body)
    .split('\n')
    .slice(0, 6)
    .map((str) => {
      return str.replace(/<\/?[^>]+(>|$)/g, '').split('\n');
    })
    .flat()
    .join(' ');
};
```

And here's the final usage.

```javascript
// src/pages/posts.astro
---
import { createExcerpt } from '../utils/create-excerpt';
import { getCollection } from 'astro:content';

const posts = await getCollection('posts');
---

<ul>
  {
    posts.map((post) => {
      const excerpt = `${createExcerpt(post.body).substring(0, 140)}...`;
      return (
        <li>
          <a href={post.slug}>
            <strong>{post.data.title}</strong>
            <p>{excerpt}</p>
          </a>
        </li>
      );
    })
  }
</ul>
```
